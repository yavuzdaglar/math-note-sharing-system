# MathNoteHub (asp.net core wep api)

This project is a premium Math Note Sharing System built with ASP.NET Core Web API and a modern, glassmorphism-inspired UI.

## Features
- **Block-based Editor**: Powerful editor with Heading, Paragraph, Image, Video, and Code blocks.
- **Media Support**: Upload photos and videos from local storage or embed from the internet (YouTube).
- **Undo/Redo**: Full history support for all editing actions.
- **Premium Design**: Dark mode support, smooth animations, and a sleek responsive interface.
- **Academic Hierarchy**: Organize notes by Categories and Courses.

## Technology Stack
- **Backend**: ASP.NET Core 8.0 Web API
- **Frontend**: ASP.NET Core MVC with TailwindCSS & Lucide Icons
- **Database**: Entity Framework Core with SQL Server
- **State Management**: Client-side block management with Undo/Redo history
